## E-Commerce Automation Project

### Project Overview
This project is about placing an order and checking out the order in the saucedemo website.

### Technologies Used
- Python
- Selenium
- Waits

### Features
- Login Automation
- Product Validation
- Cart Automation
- Checkout Automation
